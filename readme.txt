== Description ==

This is a WordPress plugin to show your Twitter badge as a widget with easy personalization. This plugin is brought to you by <a href="http://www.bloghighlight.com">Blog Highlight</a>.

Features:

1. Integrate Twitter and WordPress
2. Show Twitter badge as a WordPress widget
3. Support flash interactive widget
4. Easy personalization, no coding knowledge is needed
5. Support shortcode [twibadge]
6. Support non-widget theme

Plugin Homepage: <a href="http://www.bloghighlight.com/wordpress-plugins/twibadge/">http://www.bloghighlight.com/wordpress-plugins/twibadge/</a>

== Installation ==

1. Unzip the zip file you have downloaded (twibadge.zip)
2. Upload folder 'twibadge' to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Configure the plugin through 'TwiBadge' menu in WordPress 'Settings' page
5. Add 'TwiBadge' widget through 'Widget' menu in WordPress
6. Done, go to your blog homepage to view the twitter badge widget
7. If your theme doesn't support widget, you can add the following code to anyplace of your theme <?php showTwitterBadge(); ?>
8. You can add shortcode [twibadge] to your post or page content to show your Twitter badge

== Frequently Asked Questions ==

= Why the widget is showing me black screen? =

There are a few possibilities for this question:
1. Your Twitter username is incorrect. Please check and enter your Twitter username correctly.
2. Twitter is down. You can verify this by checking Twitter website (http://www.twitter.com).

= Why it is showing me invalid username in the option page of the administrative screen? =

Please check and enter your Twitter username carefully. Once it is successfully verified with Twitter account, this error message will not be shown.
